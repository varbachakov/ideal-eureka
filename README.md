# Mobile Yarmy

Mobile Yarmy - мобильное приложение Ярми для коротких реальных видеоисторий.

Yarmy переносит идеологию сторис в браузерный мир: история как живое публичное присутствие, реализм, возможность показать всем, где ты находишься и что делаешь. Этот репозиторий отвечает за мобильную часть этой идеи - достоверную съемку через камеру приложения.

## Продуктовая Идея

Yarmy вдохновляется философией сторис: показать живой момент быстро, понятно, с контролем автора над публикацией и быстрыми реакциями зрителей. Но ключевое отличие Yarmy - настоящий контент и доверие к происхождению видео.

История в Yarmy - это не пост и не клип ради удержания внимания. Это короткое реалистичное сообщение миру: "я здесь", "я это вижу", "я этим занимаюсь", "вот что происходит сейчас".

Зрители должны уметь реагировать на истории. Реакция - это быстрый социальный отклик на настоящий момент, а не отдельный тяжелый сценарий общения.

По реакции Yarmy стремится быть близким к Telegram Stories: быстро, понятно, без лишнего трения. По развлекательности Yarmy может смотреть в сторону легкости TikTok, но не превращаться в бесконечную ленту и не терять принцип настоящего контента.

В Telegram Stories можно выбрать фото или видео из галереи. В Yarmy нельзя: видео можно снять только через встроенную камеру приложения. Загрузка из галереи, AI-генерация, фильтры и внешний монтаж не входят в capture-flow, потому что они размывают обещание настоящего контента. Обрезка начала и конца разрешена, потому что для продукта важен факт съемки внутри приложения, а не сохранение каждой сырой секунды.

## MVP

- Длина видео: до 60 секунд.
- Запись: один непрерывный дубль без паузы.
- Первый экран: камера.
- После записи: предпросмотр, обрезка начала/конца, действия "Опубликовать" и "Удалить".
- Если пользователь не авторизован, вход запрашивается только перед публикацией.
- Просмотры, лайки и комментарии не входят в MVP, но это важное направление развития историй.

## UI Стек

- Forui - кнопки, иконки, диалоги, sheets, toasts и общая UI-система.
- Custom Yarmy UI - камера, record button, progress ring, camera overlay и trim.
- `flutter_animate` - пульсация, переходы, реакции и появление элементов.
